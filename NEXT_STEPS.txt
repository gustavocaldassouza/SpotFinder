  1. Open Xcode:
     $ open SpotFinder.xcodeproj

  2. Add files to project:
 Right-click "SpotFinder" folder     
 Select: Models, Services, ViewModels, Views, Utilities
 Check "Copy items if needed"     
 Click Add     

  3. Delete ContentView.swift:
 Move to Trash

  4. Build & Run:
 Select iOS 17+ simulator     
 Press Cmd + R     
 Grant location permission     

  5. Test:
 Map should load with your location     
 Tap "Report Spot" to test form     
 Check Settings view
